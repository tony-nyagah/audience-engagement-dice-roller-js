# 🎲 Audience Engagement Dice Roller App

A simple web app to randomly pick people from your audience during presentations. Enter names, roll the dice, and engage your crowd!

## Quick Start

```bash
npm install
npm run dev
```

Open `http://localhost:5173` in your browser.

## How to Use

1. **Add people** - Type names and hit Enter
2. **Start game** - Click the green button
3. **Roll dice** - Click to randomly pick someone
4. **Repeat** - Keep rolling or add more people

## Deploy

```bash
npm run build
```

Upload the `dist/` folder to any web host. Works on Netlify, Vercel, GitHub Pages, etc.

## Features

- Clean, mobile-friendly design
- Smooth dice animation
- No server needed
- Works offline

Perfect for presentations, classrooms, and team meetings for less than 20 people probably... 😅
